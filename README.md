# Tareas Robotica
Tareas de la materia de robotica usando OpenGL


Nota: en el tasks.json usó mi entornos MSYS UCRT64 configurado para poder usar OpenGL, por lo que al descargar es necesario cambiar la ruta por el entorno que se tenga configurado, o si se agrega en el PATH bastaría con usar g++.exe, mientras no haya conflictos con otros compiladores. Tambien es necesario modificar las rutas del tasks.json para correr el codigo y crear el ejecutable de la tarea que se desee probar.
